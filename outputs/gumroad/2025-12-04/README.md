## When to Use Each Approach

**Use the Extraction Prompts when:**
- You've created something custom in Claude AI
- You want to understand HOW it was built
- You need to modify or extend the functionality

**Use the Direct Import when:**
- You want Anthropic's official, battle-tested skills
- You're setting up Claude Code for the first time
- You prefer ready-made solutions over custom extraction
